Privacy Policy

Last updated: July 4, 2026

1. Overview

PromptGod is designed with privacy as a core principle. This Privacy Policy explains what data the Extension handles and how it is processed.

2. Data We Collect

We do not collect any personal data. PromptGod does not send any information to external servers. There is no analytics, no telemetry, and no tracking of any kind.

3. Data Processed Locally

The Extension processes the following data entirely within your browser:

  • Clipboard text: When you paste text into a supported AI chatbot, the Extension reads the pasted content to scan for sensitive patterns. This data is never transmitted.
  • Chat input text: When you click the manual Sanitize button, the Extension reads the current text in the chat input field. This data is never transmitted.
  • Vault entries: Detected sensitive values are stored in chrome.storage.local so you can review them in the popup. This storage is sandboxed to the Extension and accessible only on your device.
  • Custom rules: Any custom regex patterns or keywords you create are stored in chrome.storage.local on your device.

4. Data Sharing

We do not share any data with third parties. Since no data leaves your browser, there is nothing to share.

5. Data Retention

Vault entries and custom rules are stored locally until you choose to clear them. You can clear all vault data at any time by clicking the "Clear" button in the Extension popup. Uninstalling the Extension removes all stored data.

6. Permissions Used

The Extension requests the following Chrome permissions:

  • storage: To save vault entries, custom rules, and extension settings locally.
  • activeTab: To interact with the currently active tab for text sanitization.
  • scripting: To inject the content script that intercepts paste events.
  • clipboardRead: To read pasted text for scanning.
  • Host permissions: To run on supported AI chatbot websites only (ChatGPT, Gemini, Claude, Copilot, etc.).

7. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect any data from anyone, including children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected by the "Last updated" date at the top of this page.

9. Contact

If you have questions about this Privacy Policy, you can open an issue on the GitHub repository.
